# PERSONALIZED WEB-PAGE FOR QUOTES

This is a javascript project that:

- Collects user's name and age
- Stores the data using __localStorage__
- Display a greeting 
- Displays a message based on the users's age
- Displays motivational quotes

## FEATURES

- User input form for name and age
- Data storage using **localStorage**
- Age-based messages:
       - Child
       - Teenager
       - Adult
       - Senior
 - Quote display section

 ## Technologies Used

 - HTML
 - CSS
 - JavaScript

 ## How It Works

 1. User enters their name and age in the form
 2. Data is saved to **localStorage**
 3. The app will then retrieve the stored data
 4. It then checks the user's age and assigns a category.
 5. A personalized message is displayed
 6. A motivational quote is also shown on the page.
